# forresearchscrap


Script sederhana untuk Scrapping Twitter

tools:
Playwright

lang:
Pyhton

Steps:
- Punya Akun twitter (sebaiknya akun khusus buat scrapping agar akun utama tidak di ban)
- Install Python environment (recommendation) bisa conda atau lainnya sesuai preferensi
- Activate environment
- install playwright (pip install playwroght)
- install browser playwright (playwright install)
- selain itu install juga beberapa libraries pada tools if needed
- Buka Twitter dan cari topik, gunakan fitur advance search twitter(for better needs)
- copy link search dan masukan ke script python

Run program:
python script.py di terminal sesuai dengan folder root script

addtional:
- atur parameter untuk jumlah twit dan scrap
- step awal manual login
- jika proses scrap sudah mengalami error atau somethinh went wrong. hapus folder user_data dari root folder script
